# Веб-разработка для начинающих

Современный адаптивный лендинг для коммерческого курса по веб-разработке для детей и подростков 10–16 лет. Проект сделан на React + Vite и оформлен в стиле tech education startup: светлая тема, мягкие градиенты, glassmorphism-акценты, карточки с тенями и деликатные анимации.

## Что внутри

- адаптивный лендинг для desktop, tablet и mobile
- hero-блок с сильной визуальной подачей и CTA
- блок преимуществ курса
- пошаговая программа обучения
- примеры проектов учеников
- модульная программа курса
- отзывы
- финальный CTA-блок
- футер с контактами и быстрыми ссылками
- анимации через Framer Motion
- иконки через React Icons

## Технологии

- React
- Vite
- JavaScript
- Plain CSS
- Framer Motion
- React Icons

## Запуск проекта

```bash
npm install
npm run dev
```

## Сборка production-версии

```bash
npm run build
npm run preview
```

## Для портфолио

Проект подходит как пример коммерческого лендинга для образовательного продукта. Тексты и контакты можно быстро адаптировать под реального заказчика, а компонентная структура позволяет легко менять секции, карточки и CTA.
